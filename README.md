# k-ai : The Unified LLM CLI

**k-ai** is a next-generation, highly configurable, and user-friendly Command Line Interface for interacting with Large Language Models. It is built with a "best-of-both-worlds" philosophy, combining a rich user interface with a powerful, flexible, and provider-agnostic backend.

## Core Principles

- **0 Trust - 100% Control**: Every action is transparent and configurable.
- **0 Magic - 100% Transparency**: No hidden logic. The architecture is clear and understandable.
- **0 Hardcoding - 100% Flexibility**: Behavior is driven by a powerful `config.yaml`.

## Features

- **Provider Agnostic**: Powered by **LiteLLM**, `k-ai` supports over 100 LLM providers out of the box.
- **Rich UI**: A beautiful and interactive chat interface, featuring thinking-in-progress displays, streaming, and markdown rendering.
- **Highly Configurable**: A deep and flexible `config.yaml` allows you to control everything from model parameters to UI themes.
- **CLI & Library**: Use it as a standalone, powerful chat client or import it as a library in your own Python projects.
- **Configuration as Code**: Easily export the default configuration with `/config get` to manage your settings under version control.
